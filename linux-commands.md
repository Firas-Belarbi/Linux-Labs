# Basic Linux Commands

## Navigation
- `pwd` — Show current directory  
- `ls` — List files  
- `cd` — Change directory  

## File Operations
- `touch file.txt` — Create file  
- `mkdir folder` — Create folder  
- `rm file.txt` — Delete file  
- `cp a b` — Copy  
- `mv a b` — Move/Rename  

## Permissions
- `chmod 755 file`  
- `chown user:user file`  

## Processes
- `ps aux`  
- `top`  
- `kill PID`  

## Networking
- `ifconfig` / `ip a`  
- `ping google.com`  
- `curl ifconfig.me`
